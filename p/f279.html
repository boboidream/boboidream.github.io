Mac 下制作 Clover 启动U盘

Aug 30, 2016 | 操作系统 | 阅读

缘起：经过一个周末奋战，Hackintosh (黑苹果) 终又满血复活。此次升级经历着实惨痛。

虽啃上 macOS 10.12 Sierra 公测版，终不敌显卡驱动难题，重回旧版。但系统一旦升级，不可降级。索性格式化系统盘重装。回首望，一路坎坷，还好取得些经验、感悟，遂记录之。

千头万绪，暂且小处着手，慢慢道来，第二篇。

重装系统，制作一个启动 U 盘，算是最安全的一种方式了吧。U 盘在手，系统我有。

一、下载系统

无论白苹果、黑苹果，首先需要下载一个新系统，在应用商店搜索 OS X，你就会看到目前最新系统，点 download 下载。

如想装最新测试版系统，只需访问 Apple Beta 版软件计划 ，用苹果帐号登录，点击兑换就好啦。以后在应用商店 Purchased 标签页就可找到。

到此为止，白苹果的启动盘就制作完成了。黑苹果才需要第三步。

三、写入 Clover EFI 启动器

此部分，黑苹果才需要，且需要自己主板支持 EFI。这里分享两种方式：借助EFI Tools Clover，或直接用 Clover.pkg 安装。

第一种方式（推荐）

  1. 下载工具 EFI Tools Clover，挂载 EFI 分区与安装启动器都方便。

    工具下载：Clover Mac 版 GPT 分区 EFI Tools 安装专用工具包
    我用的版本：链接: https://pan.baidu.com/s/1gf199n1 密码: btyg

  2. 下载最新版 Clover EFI bootloader 上一条，「工具下载」里也有。

  3. 格式化 U 盘，并安装引导
    3.1. 挂载 U盘的 EFI 分区。查看U盘格式，Fat32。
    3.2 选择：格式化 EFI 分区并挂载到桌面(Fat32)
    3.3 比如我的是：disk3s1 ，那么先输入3，回车。输入1，回车
    3.4 (一)安装 Clover EFI 开机引导文件(Fat32)

  4. 此时，将 EFI 文件夹，替换为自己可以运行的文件夹。或者，自己按系统进行配置。

  5. 按自己需求改写 config.plist ，推荐使用 PlistEditor Pro 或者 clover configurator

  6. 安装 Clover.pkg 文件到 U 盘，更新 Clover 版本。

第二种方式

参考：How To Install OS X Yosemite Using Clover

  1. 下载最新版 Clover Clover EFI bootloader 第一种方式，「工具下载」的文章里也有。

  2. 挂载 U 盘，打开 terminal

  3. 一定要将破解 FakeSMC.kext 放于 /EFI/CLOVER/kexts/10.11/

  4. 其他配置，请自行查找。我将自己之前 EFI 文件夹放在 EFI 磁盘

  5. 直接安装 Clover.pkg 到 U盘

四、最后

此启动 U 盘 实为折腾黑果必备佳品，我称其为 「不死U盘」。

以后无论怎么改硬盘里的clover 配置文件都不怕，只要能进系统，一切都好办。
