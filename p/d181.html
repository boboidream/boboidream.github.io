域名邮箱这点事

Mar 3, 2018 | 最优解 | 阅读

常用 QQ 邮箱，担心泄露了 QQ 号，虽能开启字母邮箱，可前缀又太长。恰好自己有域名，决定试试域名邮箱，既个性又简洁，兼顾语义化。比如
me@zwb.io。

想实现想法有两个方案：一是自己搭建邮件服务，二是采用支持域名绑定的邮件服务商。

自己搭建邮件服务，需要服务器，也担心服务不够稳定。选择第二种方案稳妥些，尝试了最流行的国内外的邮件服务。

先说结论，域名邮箱的不二选择：Yandex

一、国内邮件服务

国内支持「域名邮箱」的服务商，有腾讯、网易、阿里。

腾讯

先说
腾讯域名邮箱，优点是直接和
QQ 绑定，相当于给你的 QQ 邮箱起个别名，可以免除管理多个邮箱带来的麻烦。假如你想要独立的邮箱，也可以使用
腾讯企业邮，有免费版。

而缺点是，免费版支持域名后缀有限，.com .cn 是可以的，而 .io .xyz .club 这些不支持，实在是遗憾。

我国内主力邮箱是 QQ 邮箱，简单使用蛮不错，也不乱发广告，良心产品，假如你的域名被支持，可以考虑。

网易

最早使用过 163 邮箱，只能说网易老没事给我发广告，实在是搞的有些恼火，又网易邮箱用户密码被拖库事件，让我完全抛弃 163。

而这次为了申请域名邮箱，再次测试
免费企业邮。先说优点，支持个性化域名，比如
.io。缺点是，必须实名认证，绑定手机号，国内应该都是，我注册找了很久都没找到删除帐号的方法。

又是一次不愉悦的体验，弃坑。

阿里

免费版企业邮箱
只需一元钱，开通。支持邮箱帐号 50 个，邮箱空间 5GB，个人网盘 2GB。看起来蛮不错。

鉴于，国内都要实名认证，又都要绑定手机号，遂放弃。

二、国外邮件服务

据说，以前 GMail 也免费过，不过现在收费，还蛮高，这两天整理邮件，才知道了 GMail
的强大之处，过滤器管理太高效，没赶上好时候。根据网友推荐，找到两个算是免费支持域名邮箱的服务商。

Zoho

zoho 免费版，支持 5G
内存，但是邮件协议只支持 SMTP 而不支持 IMAP 与 POP，界面显示，意味着没法用软件管理，so shit.

删除帐号太不友好了，我用 google 帐号登录的，要先重置 密 码。有了密码，才能删 主 机，删了主机才能删 机
构，然后才可以删除 帐 号。崩溃了。

Yandex

Yandex
注册及其友好，只需要输入用户名与密码，可以选择设置密保问题，不提供手机号，赞一个。

界面漂亮，网站用 React 写的，体验赞一个。删除帐号，只需要一步，So easy。舒服。

再看邮箱配置，简直完美

  • Required number of mailboxes (by default 1,000, to get more, contact our Support Service)
  • Unlimited-size mailbox
  • Reliable anti-spam and virus system
  • Access to mail through the web interface (from any computer) and mail programs using POP3/IMAP protocols

步骤1：配置主机

https://domain.yandex.com/domains_add/

与其他服务商配置方法大同小异，先验证域名归属，可以到自己的 DNS 解析商那，按要求设置。

步骤2：配置客户端

[https://yandex.com/support/mail/mail-
clients.html](https://link.jianshu.com?t=https%3A%2F%2Fyandex.com%2Fsupport%2Fmail
