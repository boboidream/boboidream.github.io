Automator > Service > Launch Application > no input > Keyboard > Shortcuts >Services

三、常用设置

  • 对调 Command 与 Option

    在”Change Command_L Key (Left Command)”下面选择”Command_L to Option_L”
    在”Change Command_L Key (Left Command)”下面选择”Command_L to Option_L”

  • PrintScreen 设为 Eject

  • NumLock 设为 Command+Delete 一键删除文件（mac上NumLock没用，DAS键盘对应的就是Clear键）

  • Insert 设为 FN

  • FN键盘
    • f1-f3 控制亮度
    • f2 mission control
    • f4 launchpad
    • f5-f6 键盘背光
    • f7-f9 媒体控制
    • f10-f12 音量控制
